# initStarterKit

starter-kit version

- Vue3 Vite
- TypeScript
- element-plus UI
- initCraft Dev Tool(low-code)
- 70 widget/input
- Full implement Create Form, UI, SQL, API, Report, App, export/import/restore on online
- Fully customizable

Use with https://github.com/appxq/init-server-sdk

## Install

```
cd init-starter-kit
npm i
```

## Update Render Form

```
npm install sd-render@latest
```
